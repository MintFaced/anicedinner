# ANDIG — A Nice Dinner Index Global
**The 6529 Meme Card Collector Edition · Volume I**

A formal index measuring the cost of a nice dinner across 72 cities and towns worldwide, denominated in 6529 Meme Cards.

## Deploy on Railway
- Connect your GitHub repo in Railway
- It will auto-detect `package.json` and run `npm start`

## Run Locally
```bash
npm install
npm start
```
Then open http://localhost:3000
