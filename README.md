# ANDIG — A Nice Dinner Index Global

**The 6529 Meme Card Collector Edition**

A formal index measuring the cost of a nice dinner across 69 cities and towns worldwide, denominated in 6529 Meme Cards.

## Deploy

This is a static site served via [serve](https://github.com/vercel/serve).

### Railway
- Connect your GitHub repo in Railway
- It will auto-detect `package.json` and run `npm start`

### Local
```bash
npm install
npm start
```

Then open http://localhost:3000
